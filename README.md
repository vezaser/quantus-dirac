Jak używać (skrót)

cd /root
nano install_quantus_dirac_compose_v0.4.2.sh   # wklejasz powyższy skrypt
chmod +x install_quantus_dirac_compose_v0.4.2.sh
./install_quantus_dirac_compose_v0.4.2.sh

	•	nazwę noda,
	•	adres nagród.
	•	Skrypt robi resztę.
	•	Potem: cd /root
